# FocusMind  

FocusMind es una herramienta diseñada para optimizar el estudio y la gestión de tareas basada en **la teoría de la curva del olvido de Hermann Ebbinghaus**. Con FocusMind, podrás registrar tareas y recibir recordatorios en intervalos crecientes para reforzar tu aprendizaje de manera eficiente.  

## Funcionalidades principales  
- **Registro de tareas con recordatorios espaciados**: Los intervalos de repaso se duplican progresivamente para reforzar la memoria.  
- **Ciclo de estudio extendido**: Al completar el ciclo base, la tarea se guarda para repasarla en una fecha estratégica para evitar el olvido.  
- **Calendario integrado**: Visualiza tus tareas pendientes, atrasadas y registradas en una interfaz organizada.  
- **Gestión flexible de tareas**: Elimina las tareas que ya no sean necesarias para mantener el orden.

## ¿Qué es la teoría de la curva del olvido?
La **curva del olvido** es un modelo desarrollado por Hermann Ebbinghaus que describe cómo la memoria se deteriora con el tiempo si no se refuerza. Según sus estudios, la retención de información disminuye drásticamente en las primeras horas o días después del aprendizaje, pero puede mejorar significativamente con repasos espaciados. Al aplicar esta técnica, el tiempo entre repasos aumenta progresivamente, lo que permite consolidar la información en la memoria a largo plazo. FocusMind implementa este enfoque para optimizar tu proceso de estudio y mejorar la retención del conocimiento.  

## Objetivo del proyecto  
El propósito de FocusMind es ayudarte a estudiar de manera más eficiente, evitando olvidos y mejorando la retención de información. Con un enfoque flexible y adaptativo, este programa busca ser una herramienta poderosa para estudiantes y profesionales. Este proyecto se busca que sea escrito totalmente en Java y que sea compatible en la mayoría de dispositivos.
FocusMind busca ofrecer tres modalidades para adaptarse a tu nivel de disciplina:  
- **Modo Normal** (Implementado): Recomendado para estudio relajado con recordatorios suaves.  
- **Modo Concentración** (Proximamente): Minimiza distracciones y aplica restricciones moderadas.  
- **Modo Concentración Profunda** (Proximamente): Bloqueo estricto de distracciones para máxima productividad.
